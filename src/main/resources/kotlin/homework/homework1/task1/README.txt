Входные данные:
1) Число N равное количеству компьютеров.
2) N строчек, которые представляют собой матрицу смежности 5 x 5.
3) N строчек, в каждой из которых указана ОС i-го устройства (1 <= i <= N).
4) Число P равное количеству вирусов.
5) P строчек, в каждой из которой указано имя j-го вируса (1 <= j <= P).
6) P строчек, в каждой из которых указаны номера заражённых устройств.

Ниже предоставлен пример конфигурационного файла с комментариями
5           // Количество компьютеров
0 1 1 1 0   // |
1 0 1 0 0   // |
1 1 0 1 1   // | Матрица смежности
1 0 1 0 0   // |
0 0 1 0 0   // |
Windows     // ОС устройства №1
MacOS       // ОС устройства №2
Linux       // ОС устройства №3
MacOS       // ОС устройства №4
Windows     // ОС устройства №5
2           // Количество вирусов
WannaCry    // Имя вируса №1
Petya       // Имя вируса №2
1 2         // Заражённые устройства вирусом №1
3           // Заражённые устройства вирусом №2
